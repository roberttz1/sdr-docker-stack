## 1. Fluxo de Atendimento Inteligente (O 'Cérebro')
Automação com n8n: Utilize o n8n para orquestrar toda a lógica conversacional. O objetivo não é uma árvore de decisão rígida, mas um agente que qualifica o lead, tira dúvidas, faz follow-up e realiza o agendamento de forma humanizada .
Desenvolvimento Ágil (Claude Code): Aplique o Claude Code para acelerar a criação dos prompts e a lógica de processamento da IA, reduzindo o tempo de desenvolvimento de semanas para horas .

## 2. Gestão de Dados e Infraestrutura (A 'Base')
Banco de Dados Centralizado (Supabase): Utilize o Supabase como a camada de persistência para armazenar todas as informações das conversas, agendamentos e status dos leads .
CRM/Kanban Integrado: Não entregue apenas o chat. Entregue uma interface de CRM com Kanban onde o cliente possa visualizar em qual estágio do funil de vendas cada lead está. Isso traz transparência operacional e torna o sistema indispensável para a gestão da empresa .
## 3. Interface de Inteligência (O 'Dashboard')
Dashboard Visual (Antigravity): Utilize o Antigravity para criar painéis de controle profissionais. Esses dashboards devem transformar dados brutos em insights estratégicos :
KPIs principais: Quantidade de contatos recebidos, consultas agendadas e taxa de conversão.
Insights de negócio: Gráficos que mostram os horários de pico e a comparação de atendimento (horário comercial vs. fora de hora), provando o ROI da sua automação .

## 4. Refinamento de Prompts: Ajustar o comportamento da IA com base no histórico real das conversas para eliminar novas objeções.

Monitoramento de Infraestrutura: Garantir o uptime do servidor, APIs e integrações do WhatsApp (Chatwoot, APIs, etc.).
Evolução do Sistema: Adicionar novas funcionalidades e métricas ao dashboard conforme o negócio do cliente cresce. Isso é o que transforma o seu serviço de um "custo" para um "ativo estratégico" que o cliente não quer desligar.
